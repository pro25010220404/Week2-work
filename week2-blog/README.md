# Week2 Blog

一个基于 `Vue 3 + Vite` 的课程作业项目，页面风格为简洁的博客首页。

当前项目已经完成博客首页的核心布局，包含导航栏、文章列表、分页、右侧侧边栏、页脚和响应式适配，适合用于展示小组协作开发成果。

## 项目效果

当前首页为博客风格页面，主要包含：

- 顶部导航栏
- 搜索输入框
- 文章列表卡片
- 分页组件
- 右侧侧边栏
- 页脚信息
- 移动端响应式布局

整体视觉采用暖色系博客风格，适合作为前端基础布局练习和团队协作作业。

## 当前已实现功能

### 1. 顶部导航区

- 博客 Logo 与站点名称
- 导航菜单：`首页 / 文章 / 关于`
- 搜索输入框

### 2. 文章列表区

- 使用 `ArticleCard` 组件渲染文章卡片
- 当前内置 5 篇旅游博客假数据
- 显示文章标题、日期、分类、阅读时长、摘要、标签、阅读量

### 3. 分页功能

- 使用 `Pagination` 组件实现分页
- 当前每页显示 5 篇文章
- 支持上一页、下一页和页码切换

### 4. 右侧侧边栏

- 站点信息
- 热门文章
- 分类列表
- 标签云
- 关于 / 友链

### 5. 响应式适配

- 平板端缩小侧边栏宽度
- 手机端主内容区和侧边栏改为上下布局
- 页面在较小屏幕下仍可正常浏览

## 安装依赖

```sh
npm install
```

## 启动项目

```sh
npm run dev
```

启动成功后，在浏览器打开：

[http://localhost:5173](http://localhost:5173)

```sh
npm run build
```
